This Website as a Project

While creating this website has not been particularly difficult from a technical standpoint, it still involved a fair amount of writing, as well as the use of HTML and CSS. As such, I would argue that it counts as a small project.

List of Technologies Used:

  • HTML 5
  • CSS 3
  • Git / GitHub

How it Works

It is a simple set of HTML and CSS documents hosted using GitHub Pages.

Design Decisions

  • It makes use of a high-contrast, minimalist theme to maximize readability and minimize clutter. This also makes the pages load faster.
  • Internal Navigation links are at the top, so as to make them easy to find, while also refraining from overflowing the left-hand pane at low resolutions.
  • Iframes are used to allow code reuse.

Why Do It?

I wanted a site with a simple layout on which I could showcase my personal projects.
